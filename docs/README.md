# Overview

This project demonstrates how to integrate HashiCorp Vault with Jenkins for secure secret management in CI/CD pipelines using:

***AWS IAM Authentication Method***
AWS Secrets Engine
Jenkins running on AWS EC2
Terraform for infrastructure provisioning
Dynamic AWS credential generation

Instead of storing long-term AWS Access Keys inside Jenkins, Vault dynamically generates temporary AWS credentials and securely injects them into Jenkins pipelines.
